# feedback
A feedback rating system with user and admin roles. Users log in and submit ratings from Very Poor to Excellent, with feedback required for low ratings. Admins view analytics via dashboards, filter data, export reports, and use light/dark mode. Built using React, Node.js, Express, and MySQL.
